# Changelog — it-markdown VS Code extension

## 0.2.0 — 2026-05-15

### Added

- TextMate grammar injection for `](!type:)` link destinations
- Syntax highlighting for `[!tab:]` and `[!collapse:]` block directives
- VS Code settings UI with 4 configurable options
- `it-markdown.toggleStaticOnly` command
- Editor title icon for preview command
- Theme-aware preview CSS (VS Code CSS variables)
- Marketplace metadata: icon, keywords, categories, banner
- `vsce` packaging and publish scripts
- SVG extension icon
- Proper deactivate() export
