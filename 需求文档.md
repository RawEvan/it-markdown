# it-markdown 技能改进需求文档

## 概述

当前 it-markdown 技能的安装和使用过程存在一些问题，需要进行改进以提高用户体验和稳定性。

## 问题分析

### 1. 技能安装问题

#### 问题描述
- 安装脚本没有正确地将 it-markdown 技能安装到用户的 ~/.claude/skills/ 目录下
- 安装过程中依赖安装不完整
- 没有明确的安装成功或失败的反馈

#### 根源分析
- install-skill.sh 脚本将技能安装到当前项目的 .claude/skills/ 目录下，而不是全局的用户技能目录
- npm install 可能会失败，但脚本没有处理错误

#### 改进需求
- 技能应该默认安装到 ~/.claude/skills/ 目录下
- 提供明确的安装成功/失败反馈
- 确保所有依赖都正确安装
- 支持通过命令行参数指定安装位置

### 2. 模块解析问题

#### 问题描述
- it-md-notepad 无法找到 it-markdown 模块
- 需要手动执行 npm run build 才能正常工作

#### 根源分析
- it-md-notepad/package.json 中依赖 it-markdown 为 file:..
- 项目没有预构建，导致模块解析失败

#### 改进需求
- 确保 it-markdown 模块在安装过程中已正确构建
- 提供预构建的版本，避免首次运行时需要构建
- 优化模块解析路径

### 3. 启动和使用问题

#### 问题描述
- it-md-notepad 启动后，页面可能无法正常加载
- 没有明确的启动成功反馈

#### 改进需求
- 启动时检查依赖是否完整
- 提供明确的启动状态反馈
- 添加启动超时机制
- 优化页面加载和错误处理

## 技术实现方案

### 1. 安装脚本改进

```bash
#!/usr/bin/env bash

# 全局常量
REPO_URL="https://github.com/RawEvan/it-markdown.git"
DEFAULT_INSTALL_DIR="$HOME/.it-markdown"
SKILL_NAME="it-markdown"

# 检测技能目录
if [ -d "$HOME/.claude" ]; then
    SKILL_DIR="$HOME/.claude/skills/$SKILL_NAME"
elif [ -d "$HOME/.trae" ]; then
    SKILL_DIR="$HOME/.trae/skills/$SKILL_NAME"
else
    # 默认使用 claude 目录
    SKILL_DIR="$HOME/.claude/skills/$SKILL_NAME"
    mkdir -p "$(dirname "$SKILL_DIR")"
fi

echo "=== 开始安装 $SKILL_NAME 技能 ==="

# 1. 克隆或更新仓库
if [ -d "$DEFAULT_INSTALL_DIR/.git" ]; then
    echo "1. 更新现有仓库..."
    if git -C "$DEFAULT_INSTALL_DIR" pull --quiet; then
        echo "✅ 仓库更新成功"
    else
        echo "❌ 仓库更新失败"
        exit 1
    fi
else
    echo "1. 克隆仓库..."
    if git clone --depth 1 "$REPO_URL" "$DEFAULT_INSTALL_DIR"; then
        echo "✅ 仓库克隆成功"
    else
        echo "❌ 仓库克隆失败"
        exit 1
    fi
fi

# 2. 安装依赖
echo "2. 安装依赖..."
cd "$DEFAULT_INSTALL_DIR"

if [ ! -d "node_modules" ]; then
    if npm install --silent; then
        echo "✅ 主项目依赖安装成功"
    else
        echo "❌ 主项目依赖安装失败"
        exit 1
    fi
fi

cd it-md-notepad
if [ ! -d "node_modules" ]; then
    if npm install --silent; then
        echo "✅ 编辑器依赖安装成功"
    else
        echo "❌ 编辑器依赖安装失败"
        exit 1
    fi
fi

# 3. 构建项目
echo "3. 构建项目..."
cd "$DEFAULT_INSTALL_DIR"
if npm run build; then
    echo "✅ 项目构建成功"
else
    echo "❌ 项目构建失败"
    exit 1
fi

# 4. 安装技能链接
echo "4. 安装技能链接..."
mkdir -p "$(dirname "$SKILL_DIR")"

if [ -L "$SKILL_DIR" ]; then
    rm "$SKILL_DIR"
fi

ln -sf "$DEFAULT_INSTALL_DIR/skills/$SKILL_NAME" "$SKILL_DIR"

echo "=== 安装成功 ==="
echo ""
echo "技能位置: $SKILL_DIR"
echo "编辑器位置: $DEFAULT_INSTALL_DIR/it-md-notepad"
echo ""
echo "使用方法:"
echo "  node $DEFAULT_INSTALL_DIR/it-md-notepad/bin/it-md-notepad.mjs <file.md>"
echo ""
echo "或通过浏览器访问:"
echo "  http://localhost:5173"
```

### 2. 启动脚本改进

```javascript
// bin/it-md-notepad.mjs 改进版

import { createServer } from 'http';
import { fileURLToPath } from 'url';
import { dirname, join, resolve } from 'path';
import { readFileSync, existsSync } from 'fs';

const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);

// 检查依赖是否完整
function checkDependencies() {
    const itMdPath = join(__dirname, '..', '..', 'dist', 'index.js');
    if (!existsSync(itMdPath)) {
        console.error('❌ it-markdown 模块未找到，请先运行 npm run build');
        return false;
    }
    return true;
}

// 启动服务器
function startServer() {
    console.log('🚀 启动 it-md-notepad...');
    
    const app = createServer((req, res) => {
        // 处理文件路径
        const parsedUrl = new URL(req.url, 'http://localhost');
        const filePath = parsedUrl.searchParams.get('file');
        
        if (req.method === 'GET' && parsedUrl.pathname === '/api/file' && filePath) {
            // 读取文件
            const absPath = resolve(filePath);
            if (!existsSync(absPath)) {
                res.writeHead(404, { 'Content-Type': 'application/json' });
                res.end(JSON.stringify({ error: 'File not found' }));
                return;
            }
            
            try {
                const content = readFileSync(absPath, 'utf8');
                res.writeHead(200, { 'Content-Type': 'application/json' });
                res.end(JSON.stringify({ content }));
            } catch (err) {
                res.writeHead(500, { 'Content-Type': 'application/json' });
                res.end(JSON.stringify({ error: err.message }));
            }
            return;
        }
        
        // 静态文件服务
        let staticPath = '';
        if (parsedUrl.pathname === '/') {
            staticPath = join(__dirname, '..', 'index.html');
        } else {
            staticPath = join(__dirname, '..', parsedUrl.pathname.slice(1));
        }
        
        if (!existsSync(staticPath)) {
            res.writeHead(404, { 'Content-Type': 'text/plain' });
            res.end('Not found');
            return;
        }
        
        // 设置正确的 Content-Type
        let contentType = 'text/plain';
        if (staticPath.endsWith('.html')) contentType = 'text/html';
        if (staticPath.endsWith('.js')) contentType = 'application/javascript';
        if (staticPath.endsWith('.css')) contentType = 'text/css';
        if (staticPath.endsWith('.json')) contentType = 'application/json';
        if (staticPath.endsWith('.png')) contentType = 'image/png';
        if (staticPath.endsWith('.jpg') || staticPath.endsWith('.jpeg')) contentType = 'image/jpeg';
        if (staticPath.endsWith('.svg')) contentType = 'image/svg+xml';
        
        res.writeHead(200, { 'Content-Type': contentType });
        res.end(readFileSync(staticPath));
    });
    
    // 尝试在 5173 端口启动
    app.listen(5173, '127.0.0.1', () => {
        console.log('✅ it-md-notepad 正在运行');
        console.log('');
        console.log('📖 访问地址: http://localhost:5173');
        console.log('');
        if (parsedUrl.searchParams.get('file')) {
            console.log('📄 打开文件: http://localhost:5173?file=' + encodeURIComponent(filePath));
        }
    }).on('error', (err) => {
        if (err.code === 'EADDRINUSE') {
            console.error('❌ 端口 5173 已被占用，请检查是否有其他实例在运行');
        } else {
            console.error('❌ 启动失败:', err.message);
        }
        process.exit(1);
    });
    
    // 处理信号
    process.on('SIGINT', () => {
        console.log('\n🔴 正在停止服务器...');
        app.close(() => {
            process.exit(0);
        });
    });
}

// 主程序
if (!checkDependencies()) {
    process.exit(1);
}

const args = process.argv.slice(2);
startServer(args[0]);
```
