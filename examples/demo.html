This is the Overview tab. iMD extends standard CommonMark with a small set of interactive widgets.

Install with npm:

npm install it-markdown

Then import and render:

import { renderImdToHtml } from "it-markdown";
const html = renderImdToHtml(source);

See SPEC.md for the full iMD grammar specification. Widgets use the link-style syntax: [label](!type:key=val)
