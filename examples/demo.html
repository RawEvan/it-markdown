This is the Overview tab. These extensions extend standard CommonMark with a small set of interactive widgets.

Install with npm:

npm install it-markdown

Then import and render:

import { renderItMarkdownToHtml } from "it-markdown";
const html = renderItMarkdownToHtml(source);

See SPEC.md for the extension grammar specification. Widgets use the link-style syntax: [label](!type:key=val)
